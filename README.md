# Ticketing Microservices System

Microservices-based ticket management system inspired by Jira, developed using Spring Boot, Spring Cloud, Eureka, API Gateway, OpenFeign, MySQL, and Docker.

## Features

* Microservices architecture
* Service discovery with Eureka
* API Gateway routing and filtering
* Distributed configuration management
* Ticket management workflows
* User, team, and role management
* Assignment and notification services
* Activity logging and comments
* REST API communication between services
* Dockerized services

## Microservices

* User Service
* Ticket Service
* Assignment Service
* Notification Service
* Activity Service
* API Gateway
* Eureka Server
* Config Server

## Technologies

* Java 21
* Spring Boot
* Spring Cloud
* OpenFeign
* Eureka
* API Gateway
* MySQL
* Docker
* Maven

## Architecture

The system follows a distributed microservices architecture where services communicate through REST APIs and service discovery mechanisms. API Gateway is used for centralized routing and request filtering, while Eureka handles service registration and discovery.

## Purpose

Academic project focused on distributed systems, microservices communication, database-driven workflows, and enterprise application architecture.
